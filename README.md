# 1 dimensionales Array

Definition:
          Hauptsächlich sind Eindimensionele Arrays eine Liste.
          Dieser Array sind zusammengefügte Objekte einer Variable.
          Bzw. es sind mehrere Werte die den selben Variablentyp haben.
